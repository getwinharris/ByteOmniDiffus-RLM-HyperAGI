# AGENTS.md

## Purpose

Own repository maintenance tooling.

## Ownership

This subtree owns scripts that inspect, index, verify, or maintain repository structure. It does not own product runtime code.

## Local Contracts

- Keep tools dependency-free unless the root contract justifies a dependency.
- Tools must be deterministic and repeatable.
- Generated outputs must state which tool generated them.
- `project_map.mmd` is generated by `tools/generate_project_map.py` and must not be hand-edited.

## Work Guidance

- Prefer standard-library Python for repository maintenance scripts.
- Keep command interfaces simple.
- Support `--check` for generated artifacts when practical.

## Verification

```bash
python3 tools/generate_project_map.py --check
```

## Child DOCS-INDEX Index

- `generate_project_map.py` — scans repository files and writes/verifies `project_map.mmd`.
